# 网络安全靶场系统 - 毕业设计项目

## 项目概述
网络安全靶场系统是一个现代化的网络安全攻防演练平台。该系统提供了完整的攻击模拟、防御策略实施、实时监控和态势感知功能，旨在为网络安全专业人员提供一个高效的学习和训练环境。

## 系统特性
- **多样化的攻击模拟**: 支持多种常见攻击类型的模拟
- **多层次防御系统**: 提供防火墙、WAF、IDS/IPS等防御机制
- **实时态势感知**: 提供实时的攻击/防御状态监控
- **可视化展示**: 丰富的图表和拓扑图展示系统状态
- **靶场环境管理**: 支持动态创建和管理靶场环境
- **日志分析**: 完整的日志记录和分析功能
- **用户友好的界面**: 现代化的Web界面，易于操作

## 系统架构
系统采用前后端分离架构：
- **前端**: Vue.js + Element Plus + ECharts
- **后端**: Python Flask + SQLite
- **容器化**: Docker用于靶场环境管理
- **数据库**: SQLite轻量级数据库存储系统数据

## 功能模块

### 1. 攻击模拟
- **多样化攻击**: 支持16种常见攻击类型
- **参数化配置**: 可自定义攻击参数和强度
- **实时监控**: 攻击执行状态实时跟踪

### 2. 防御系统
- **多层防御**: 防火墙、WAF、IDS/IPS等
- **智能响应**: 根据威胁等级自动调整策略
- **态势感知**: 实时监控系统安全状态

### 3. 靶场管理
- **环境创建**: 动态创建靶场环境
- **容器管理**: Docker容器生命周期管理
- **资源监控**: 靶场资源使用情况监控

### 4. 日志分析
- **全面记录**: 记录所有攻击和防御活动
- **实时分析**: 日志实时分析和威胁检测
- **可视化展示**: 图表形式展示日志趋势

## 技术栈
- **前端技术**:
  - Vue 3: 前端框架
  - Element Plus: UI组件库
  - ECharts: 数据可视化
  - Axios: HTTP客户端
  - Vue Router: 路由管理

- **后端技术**:
  - Python: 主要开发语言
  - Flask: Web框架
  - SQLite: 数据库
  - Docker: 容器化
  - APScheduler: 任务调度

## 安装部署

### 环境要求
- Python 3.8+
- Node.js 14+
- Docker
- Git

### 安装步骤
1. 兢隆项目
   ```bash
   git clone <repository-url>
   cd cyber-range
   ```

2. 安装后端依赖
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

3. 安装前端依赖
   ```bash
   cd ../frontend
   npm install
   ```

4. 启动系统
   ```bash
   # 启动后端
   cd backend
   python run.py
   
   # 启动前端
   cd frontend
   npm run dev
   ```
