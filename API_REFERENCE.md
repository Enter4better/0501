## 概述

### 基础信息
- **后端框架**: Flask
- **认证方式**: JWT (JSON Web Token)
- **API前缀**: `/api`
- **返回格式**: JSON

### 通用响应格式
```json
{
  "status": "success" | "error",
  "msg": "消息内容",
  "data": {}  // 具体数据
}
```

### 认证说明
除登录、注册接口外，其他接口均需要在请求头中携带JWT Token：
```
Authorization: Bearer <token>
```

## 认证模块

**路由前缀**: `/api/auth`

### 已实现接口

| 接口 | 方法 | 路径 | 说明 | 状态 |
|------|------|------|------|------|
| 用户注册 | POST | `/register` | 注册新用户 | ✅ 已实现 |
| 用户登录 | POST | `/login` | 用户登录获取Token | ✅ 已实现 |
| 获取用户信息 | GET | `/profile` | 获取当前用户信息 | ✅ 已实现 |
| 更新用户信息 | PUT | `/profile` | 更新用户信息 | ✅ 已实现 |
| 用户登出 | POST | `/logout` | 用户登出 | ✅ 已实现 |
| 修改密码 | POST | `/change-password` | 修改用户密码 | ✅ 已实现 |
| 获取用户列表 | GET | `/users` | 获取所有用户(管理员) | ✅ 已实现 |

### 接口详情

#### POST `/register` - 用户注册
**请求参数**:
```json
{
  "username": "用户名(3-20位)",
  "password": "密码(至少8位,含大小写字母和数字)",
  "email": "邮箱地址",
  "role": "角色(user/admin,可选)"
}
```
**响应示例**:
```json
{
  "status": "success",
  "msg": "注册成功",
  "user": {
    "user_id": "xxx",
    "username": "test",
    "email": "test@example.com",
    "role": "user"
  }
}
```

#### POST `/login` - 用户登录
**请求参数**:
```json
{
  "username": "用户名",
  "password": "密码"
}
```
**响应示例**:
```json
{
  "status": "success",
  "msg": "登录成功",
  "token": "JWT Token",
  "user": {
    "user_id": "xxx",
    "username": "test",
    "email": "test@example.com",
    "role": "user",
    "last_login": "2026-04-24T10:00:00"
  }
}
```
